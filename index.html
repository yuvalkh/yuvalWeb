Three main diffrences between web1,web2,web3


in compare to web2 and web3, web1 had limited interactions and just works as an information portal to the viewers.
Moreover, you can use crawlers in web2 and web3, but not in web1.
In addition, web3 will be a semantic web where many things will be connected to each other '

Use case of query search "side effects of medicines"

in web1: we can seek some information about that(ofc only by if we know the url of the site) but if we want we can't post a question about these effects, in other words, we can only read that information
in web2: we can search in search engine and then find many links and urls that the search engine tells us they relate to the query by the words there or by tags.
in web3: we can search too in search engine but this time we will get pages that has good connection to the query and we will probably get better results from web2.
